## <u>Bug Fixes</u>

- Fixed `lager update` command to correctly update Lager Box status after updates
- Fixed `lager update --all` to correctly sync all Lager Boxes

## <u>Improvements</u>

- Cleaned up `--help` output across CLI commands for better readability

## <u>Installation</u>

To install this version:

```bash
pip install lager-cli==0.3.15
```

To upgrade from a previous version:

```bash
pip install --upgrade lager-cli
```
